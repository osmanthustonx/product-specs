# Todo Image Attachment Feature - Proposal

## Summary
為 todo 項目新增圖片附加功能，讓用戶能透過視覺化參考提升任務清晰度。

## Motivation
用戶在處理各類型 todo 時，常需要附加參考資訊：
- 購物清單需要商品圖片
- 家務任務需要現場照片/進度證明
- 設計工作需要參考稿或設計圖
- DIY 專案需要成品或步驟圖片

目前 todo 應用缺乏圖片附加功能，導致用戶需要在其他地方（手機相冊、備忘錄等）管理相關圖片，增加應用間切換的成本。

## Scope

### In Scope
- 單個 todo 可附加 1 張圖片
- 支援 JPG、PNG、WebP 格式
- 圖片檔案大小限制 5 MB
- 清單頁面顯示圖片縮圖
- 詳情頁面顯示完整尺寸圖片
- 編輯 todo 時支援上傳/替換/刪除圖片

### Out of Scope
- 圖片編輯功能（裁切、濾鏡等）
- 批量上傳多張圖片
- 圖片標籤或分類
- 圖片分享功能
- 人工智慧圖片辨識

## Expected Outcomes
✅ 提升任務視覺化程度，降低用戶認知負荷
✅ 減少應用外管理輔助資料的需求
✅ 提高用戶 engagement（在應用內完成更多任務流程）
✅ 為未來擴展（多圖、圖片庫等）奠定基礎
